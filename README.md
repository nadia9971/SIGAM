breve introduccion:

este módulo automatiza la creación de turnos
médicos, eliminando el uso de libretas físicas, procesos lentos e
inconformidades.
Objetivo de las interfaces: facilitar un registro siendo más rápido, fácil
y legible, garantizando el orden de llegada y proporcionando tiempos estimados
de espera para mejorar la experiencia de personas de 3ra edad o con
discapacidades.
incluye también un login donde cada personal entra a su area correspondiente mediante un prefijo (Medico, Administrador o Recepcion). evitando asi perdida o robo de datos.
